# Generative_AI_Research

## Project Overview

This repository contains all research materials for the study **"Using Generative AI Tools – Boon or Bane"**, including literature reviews, survey data, interview transcripts, analysis scripts, draft reports, and supporting materials. The project investigates the benefits and drawbacks of generative AI tools in contemporary research and professional contexts.

## File Naming Convention

All files in this repository follow the **HMS (Harvard Medical School) file naming convention**:

```
[YYYYMMDD]_[Project]_[FileType]_[v##].ext
```

| Element     | Description                                      | Example              |
|-------------|--------------------------------------------------|----------------------|
| `YYYYMMDD`  | Date created or last modified (ISO 8601 format)  | `20260429`           |
| `Project`   | Project abbreviation                             | `GenAI`              |
| `FileType`  | Brief descriptor of file content                 | `SurveyData-Raw`     |
| `v##`       | Version number with leading zero                 | `v01`, `v02`         |

**Rules:**
- Use underscores (`_`) to separate metadata elements
- Use dashes (`-`) within a single element (e.g., `SurveyData-Raw`)
- No spaces, no special characters (`~ ! @ # $ % ^ & *`)
- File names should be 40–50 characters maximum
